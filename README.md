### Specification 3
* After we pass multiple arguments, when we keep changing directories, the changed directory is now the previous directory.
* The start of a file does not have `-` character.

### Specification 4
* I am just coloring the name of the file with the specified color codes. 

* A command like `reveal -lalalas` will still be considered correct.

* In the `-a` flag, all files have been colored blue.

### Specification 5
* If there is a command execute by `log execute index`, that would not be further stored in the logfile.

* The logfile is a text file.

### Specification 6
* The executable command is an actual UNIX command.

* If there is an invalid background command, the system tries to run it, but fails with exit status 1.
